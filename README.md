# My Portfolio Website

A simple multi-page portfolio website built with HTML and CSS.

## Pages
- About Me (index.html)
- Portfolio (portfolio.html)
- Hobby (hobby.html)
- Contact (contact.html)

## What I learned
- Basic HTML structure (headings, paragraphs, lists)
- Linking pages using anchor tags
- Using one CSS file to style multiple pages

## How to run
Download the files and open `index.html` in a browser.
